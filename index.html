Installation und Verwendung

Die Verwendung des Burning-Series-Downloader ist ziemlich simpel:

  • Kopiere den passenden Geckodriver in den entpackten Ordner
  • Erstelle wie folgt ein Objekt und rufe danach die Funktion 'get_serie()' auf

from main import crawl

c = crawl('https://bs.to/serie/BSLINK')

c.get_serie()

Danach werden alle Staffeln in ein mit dem Namen der Serie erstelltes Verzeichnis kopiert

Alternativ kann die grafische Oberfläche, die mit Tkinter implementiert wurde, deaktiviert werden:

from main import crawl

c = crawl('https://bs.to/serie/BSLINK')

c.gui = "off"

c.get_serie()
